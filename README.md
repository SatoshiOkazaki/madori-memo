# 間取りメモ

iPad + Apple Pencil で間取り図を簡易作成するWebアプリ（PWA）。
ビルド工程なし・外部依存ゼロ・完全オフライン動作。

## 起動方法

```sh
cd madori-app
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開く。

iPadで使う場合は、Mac と同じWi-Fi上で `http://<MacのIPアドレス>:8000` を開く。
ホーム画面に追加すると全画面のPWAとして動作する
（Service Worker / 共有シートをフルに使うには HTTPS または localhost が必要）。

## 操作

- **Apple Pencil（ペン）**: 描画・配置・選択・消しゴムなど、選択中ツールの操作
- **指1本**: キャンバスのパン
- **指2本**: ピンチズーム（0.25〜4倍）
- **壁モード**: 壁線（自動で直線補正+0°/45°/90°スナップ）、ドア/引き戸/窓の配置、選択（移動・回転90°・削除）、消しゴム
- **メモモード**: フリーハンド描画（黒/赤/青/緑、細/太）、消しゴム
- データは1秒デバウンスで localStorage に自動保存され、次回起動時に復元される
- 「画像で保存」で図面全体をPNG化（iPadでは共有シート、その他はダウンロード）
